# vue-dashboard

A project to explore vue2 with some packages. The packages include:

- Vuetify
- I18n
- Vue
- Router
- Apexchart

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
